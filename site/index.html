What ForkSure Does

ForkSure is a Python CLI that gathers repository evidence from GitHub and local project files. It helps reviewers inspect fork provenance, attribution, license drift, suspicious name similarity, exact code similarity, and basic safety signals.

Why It Exists

Public repositories are easy to fork, rename, mirror, and imitate. ForkSure keeps the first pass practical: collect evidence, show the risk signals separately, and leave conclusions to a human reviewer.

Core features

Focused Signals, Readable Output

Fork Audits

List forks and compare license and README attribution against the source repository.

Imposter Scan

Find same-name and similar-name repositories, enrich candidates, and produce HTML evidence reports.

Rare-String Discovery

Search for uncommon README phrases to surface additional repositories worth reviewing.

Repository Compare

Compare two repositories using metadata, license, README attribution, and optional clone similarity.

Safety Wrappers

Run local checks for unsafe scripts, secrets, SAST, dependency hygiene, and OSV when tools are available.

HTML Reports

Generate static reports that are easier to review than a wide terminal table.

Example CLI usage

Run Targeted Scans

python -m forksure.cli forks OWNER/REPO --audit-license --audit-readme
python -m forksure.cli imposters OWNER/REPO --rare-strings --html
python -m forksure.cli compare OWNER/REPO OTHER/REPO --similarity --html
python -m forksure.cli security audit .

Risk Model

ForkSure separates risk signals instead of flattening everything into one vague score. Compare reports show overall risk alongside name/imposter, README attribution, license, code similarity, and security categories.

HTML Reports

Imposter and repository compare commands can write self-contained HTML reports with evidence, links, reasons, and neutral language for manual review.

Installation

Development Setup

python -m pip install uv
uv sync
python -m pytest -q

Set GITHUB_TOKEN for higher GitHub API limits when scanning public repositories.
